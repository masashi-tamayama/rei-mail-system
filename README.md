# Rei Mail System

Rei Mail System はメール送信サービスを構築するためのシステムです。  
このシステムは、Laravelフレームワークを使用して構築されており、以下の主要な機能を提供します：

- 会員登録とログイン
- メールリストのアップロード（CSV対応）
- メールの作成（WYSIWYGエディタ対応）
- メールの一括送信（非同期処理）
